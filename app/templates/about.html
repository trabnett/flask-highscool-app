About:

Welcome to TRHS, a project to simulate a highschool website created using Flask and Postgress. This project uses many common features of Flask, Postgress and Flask-Bootstrap.

To best experience the app, log in as either a teacher or student. You can either sign up or use one of the following example accounts:

  • exampleteacher@trhs.com, password: asdf
  • examplestudent@trhs.com, password: asdf

The main purpose of this app was to create relational data between students and teachers using SQLAlchemy.

For Teachers, this means:

  • Teachers can create Courses and add or remove Students to them.
  • Teachers can create Sports, however, only Students can add or remove themselves from Sports
  • Teachers can create Tests and record StudentTest Scores
  • Teachers can delete their account, removing all records of their Courses, Sports, and Tests

For Students, this means:

  • Students can enroll or unenroll from Sports
  • Students can see thier individual Test Scores and Course Averages
  • Students can delete their account, removing all records of their Student Courses, Student Tests and Student Sports

Responsive styling is done using Flask-Bootstrap. Forms are created using WTForms. Scripting done using Javascript and jQuery.
